Objectif du Dashboard

Deux types de sondes sont utilisés

  • Les sondes automatiques (KAPTA)
  • Les points de surveillance (PSV)

Les principaux objectifs de ce dashboard sont les suivants :

  • Visualisation des données dépassant le seuil de chlore sélectionné
  • Affichage du nombre d’occurrence des dépassements de seuil
  • Visualisation d’un graphique pluviométrique sur une période sélectionnée
  • Affichage d'une carte interactive des canalisations de Nice.
  • Visualisation global des concentrations de chlore sur une période sélectionnée (KAPTA / PSV)
